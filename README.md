# 🛒 Проект Маркетплейс

## 📝 Описание проекта

**Проект Маркетплейс** — это веб-приложение для размещения и поиска объявлений. Платформа позволяет пользователям публиковать свои товары и услуги, а также находить интересные предложения от других пользователей.

## ✨ Основные возможности

- ✅ Регистрация и авторизация пользователей
- ✅ Создание, просмотр, редактирование и удаление объявлений
- ✅ Поиск объявлений по названию
- ✅ Привязка объявлений к автору
- ✅ Удобный пользовательский интерфейс

---

## User Stories

### Регистрация пользователя
**Как** новый пользователь
**Я хочу** зарегистрироваться на платформе
**Чтобы** создавать и просматривать объявления

### Создание объявления
**Как** авторизованный пользователь
**Я хочу** разместить объявление о товаре
**Чтобы** другие пользователи могли его найти

### Поиск объявлений
**Как** любой пользователь
**Я хочу** искать объявления по названию
**Чтобы** быстро находить нужные товары

### Редактирование объявления
**Как** автор объявления
**Я хочу** редактировать или удалять своё объявление
**Чтобы** обновлять информацию о товаре

**Дополнительно:**

## 🔑 Переменные окружения

Создайте файл `.env` в корне проекта:

```env

SECRET_KEY=...
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
```

## 🚀 Как запустить проект

### Локальный запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/happygame406/alpha_projekt.git
   cd alpha_projekt
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Примените миграции:**
   ```bash
   alembic upgrade head
   ```

4. **Запустите приложение:**
   ```bash
   uvicorn app.main:app --reload
   ```

5. **Откройте в браузере:**
   - **API документация:** [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
   - **Фронтенд:** http://127.0.0.1:8000
