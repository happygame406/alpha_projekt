## Роли пользователей

| Роль       | Описание                                   | Права |
|------------|--------------------------------------------|-------|
| **Guest**  | Неавторизованный посетитель                | Просмотр объявлений |
| **User**   | Зарегистрированный пользователь            | Создание, просмотр, редактирование и удаление **своих** объявлений |

---

## User Stories

1. **Как** Гость  
   **Я хочу** просматривать все объявления на сайте  
   **Чтобы** находить интересующие меня товары.

2. **Как** Пользователь  
   **Я хочу** зарегистрироваться и войти в аккаунт  
   **Чтобы** получить возможность создавать свои объявления.

3. **Как** Пользователь  
   **Я хочу** создать новое объявление  
   **Чтобы** выставить товар на продажу.

4. **Как** Пользователь  
   **Я хочу** просматривать и управлять своими объявлениями  
   **Чтобы** редактировать или удалять их.

5. **Как** Пользователь  
   **Я хочу** редактировать своё объявление  
   **Чтобы** обновлять информацию о товаре.

6. **Как** Пользователь  
   **Я хочу** удалить своё объявление  
   **Чтобы** убрать его с площадки.
