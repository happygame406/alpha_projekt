# 🛒 Marketplace

**Бета-версия проекта** — удобная веб-платформа для размещения и просмотра объявлений.

## 👥 Роли и права пользователей

| Роль       | Описание                              | Возможности |
|------------|---------------------------------------|-----------|
| **Guest**  | Неавторизованный пользователь        | 👀 Просмотр всех объявлений |
| **User**   | Зарегистрированный пользователь      | ➕ Создание, ✏️ редактирование и 🗑️ удаление **своих** объявлений |

## 📋 User Stories (1 спринт)

1. **Как** Гость  
   **Я хочу** просматривать все объявления  
   **Чтобы** находить интересующие товары. 👀

2. **Как** Пользователь  
   **Я хочу** зарегистрироваться и войти в аккаунт  
   **Чтобы** создавать свои объявления. 🔑

3. **Как** Пользователь  
   **Я хочу** создать новое объявление  
   **Чтобы** выставить товар на продажу. ➕

4. **Как** Пользователь  
   **Я хочу** управлять своими объявлениями  
   **Чтобы** редактировать или удалять их. ⚙️

## ✨ Основные возможности
- Создание и просмотр объявлений
- Регистрация и авторизация
- Валидация данных
- Работа с базой данных

## 🛠️ Стек технологий
- **Backend**: FastAPI ⚡
- **БД**: SQLModel + SQLite 🗄️
- **Миграции**: Alembic 🔄
- **Тестирование**: pytest ✅
- **Frontend**: HTML + JavaScript (минимальный)

## 📁 Структура проекта
```
app/
├── models/          # User, Item
├── routes/          # Эндпоинты
├── database.py
├── main.py
alembic/             # Миграции
tests/               # Тесты
frontend/
```

## 🔑 Переменные окружения (.env)
```env
DATABASE_URL=sqlite:///./marketplace.db
SECRET_KEY=your_super_secret_key_change_me
ALGORITHM=HS256
```

## 🚀 Как запустить проект

```bash
# 1. Активировать виртуальное окружение
venv\Scripts\activate

# 2. Применить миграции
alembic upgrade head

# 3. Запустить сервер
uvicorn app.main:app --reload
```

**Документация API:** [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) 📖
